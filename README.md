# BtCarduino
Bluetooth control arduino (AVR) car

## Requirement:
### Hardware:
- Arduino Nano V3.0 ATMEGA328
- bluetooth: HT-05
- motor drive: L298N 馬達驅動板模塊 (C6B1)
- motor x2: TT馬達 減速比1:48
